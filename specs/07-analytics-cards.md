# 07 — Analytics Cards

> Inline chart cards that appear within the chat thread, generated by the AI assistant. Always appear in a 2-column grid below the "Generated Analytics" label.

---

## Section Label

- **Display:** flex row, align-items center, gap 8px
- **Left margin:** 48px (aligns with message body, past the avatar)
- **Margin bottom:** 16px
- **Icon:** Sparkles, 18×18px, color blue-600 (#1570EF) in light / brand-400 (#B692F6) in dark
- **Text:** "Generated Analytics" — 16px, weight 500, same color as icon

## Grid

- **Display:** grid, 2 columns equal width
- **Gap:** 14px
- **Left margin:** 48px (aligns with message text)

## Card

- **Background:** white (light) / #18181E (dark)
- **Border:** 1px solid gray-300 (light) / rgba(255,255,255, 0.1) (dark)
- **Border radius:** 8px (radius-md) in light / 12px (radius-lg) in dark
- **Padding:** 20px
- **Shadow:** shadow-sm (light) / deeper shadow (dark)
- **Hover (dark mode only):** border-color shifts to brand-800, add subtle purple glow

### Header

- **Display:** flex row, justify-content space-between, align-items flex-start
- **Margin bottom:** 20px

#### Left Side

- **Label:** font 12px, weight 500, color gray-400 (light) / #71717A (dark)
- **Value:** font 26px (display area), weight 600-700, color gray-900 (light) / #F4F4F5 (dark), letter-spacing -0.5px, margin-top 6px

#### Trend Badge

- **Display:** flex row, align-items center, gap 4px
- **Padding:** 2px 10px (light) / 3px 10px (dark)
- **Border radius:** 9999px (full pill)
- **Background:** rgba(82,196,26, 0.1) for positive trends
- **Text:** 12px, weight 500-600, color #52c41a (green for up)
- **Icon:** TrendingUp, 13-14px

### Chart Area

- **Height:** 160px (fixed)
- **Width:** 100% of card content area
- **Type:** Line chart (left card), Area chart (right card)

#### Chart Styling

- **Line/area stroke color:** #1677FF (blue) in light Ant theme / #9E77ED (brand-500) in dark
- **Area fill:** rgba(22,119,255, 0.08) in light / rgba(158,119,237, 0.08) in dark
- **Line width:** 2.5px
- **Data point circles:** 4px radius, same color as line
- **Grid lines:** #f0f0f0 in light / rgba(255,255,255, 0.04) in dark, dashed for vertical, solid for horizontal
- **Axis labels:** 11-12px, color gray-400 in light / #52525B in dark
- **X-axis labels:** Mon, Tue, Wed, Thu, Fri, Sat, Sun (centered under grid lines)
- **Y-axis labels:** 0, 250, 500, 750, 1000 (right-aligned, to the left of the chart area)
